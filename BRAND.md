# PMJ Jewels — Brand Guidelines (Catalogue)

## Official logos (transparent PNG)

| File | Layout | Use |
|------|--------|-----|
| `assets/pmj-logo-stacked.png` | Vertical lockup (white mark) | Cinematic loading, vault intro, dark header, mobile footer |
| `assets/pmj-logo-horizontal.png` | Horizontal bar (navy mark) | Login, light header, desktop footer |

**Tagline:** Celebrate For Ever

Both files have **transparent backgrounds** — no black box behind the mark.

## Where each logo appears

| Screen | Logo |
|--------|------|
| Loading / vault cinematic | Stacked white on dark (`cinematic-brand.css`) |
| Login form | Horizontal navy on white card |
| Header dark theme | Stacked white |
| Header light theme | Horizontal navy |
| Footer mobile / dark | Stacked white |
| Footer desktop + light | Horizontal navy |

## Stylesheets

- `css/brand.css` — logo sizing & theme swap
- `css/cinematic-brand.css` — dark cinematic stage + gold halo
- `css/login-gate.css` — light login card
- `css/gradient-buttons.css` — primary CTA gradient rings

## Do

- Use only the two official PNGs above.
- Keep aspect ratio; size with height or max-width classes.
- Alt text: `PMJ Jewels — Celebrate For Ever`.

## Don’t

- Add backgrounds, borders, or boxes behind logos.
- Tint, filter, or recolour the artwork.
- Use old text-only “PMJ Fine Jewellery” lockups in customer UI.

## Copy

- Brand name: **PMJ Jewels**
- Tagline: **Celebrate For Ever**
